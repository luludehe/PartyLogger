# PartyLogger

A SvelteKit project for logging and managing party tickets.

## Project Setup

## Via Docker (Easiest)

### Prerequisites

- Docker
- Docker Compose

### Running the project

```bash
docker-compose up
```
The server will be running on `http://0.0.0.0:3000`.

## Manual Setup

### Prerequisites

- Node.js 18.x
- npm

### Installing the dependencies

```bash
npm install
```

### Building the project

```bash
npm run build
```

### Running the project

```bash
npm build
```
